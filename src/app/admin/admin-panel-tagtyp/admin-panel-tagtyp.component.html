Typen der Sonstigen Informationen (TAGs)

Damit können beliebige zusätzliche Informationen bei den Arbeitsplätzen eingetragen werden.

Flag = 1 TAG hat keinen Text. Beim Arbeitsplatz wird nur der TAG gesetzt ohne weitere Infos.

Parameter - zur Zeit ohne Funktion.

Die Zuordnung zur Arbeitsplatz-Kategorie kann nicht nachträglich geändert werden.
