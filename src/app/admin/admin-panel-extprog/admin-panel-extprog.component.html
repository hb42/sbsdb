Externe Programme

Hier werden die externen Programme den Arbeitsplatz-Typen zugeordnet. In der FAT-Client-Anwendung können diese Programme dann für den entsprechenden AP-Typ ausgeführt werden

Schlüssel - Name, der dem Powershell-Script, das die Programme startet, übergeben wird.

Parameter - Zusätzliche Parameter, die bei Bedarf an das Powershell-Script übergeben werden.

Flag = 1 Dieses Programm wird allen Benutzern angezeigt. Programme mit Flag=0 werden nur Admins angezeigt.
